# 🏦 Banka Yönetim ve Otomasyon Sistemi
C# ve MS SQL Server ile geliştirilmiş, müşteri kaydı, bakiye yönetimi, vadeli/vadesiz hesap türlerine göre faiz işlemlerini yönetme ve temel finansal işlemleri (havale, para yatırma/çekme) içeren banka otomasyon sistemi.

# Öne Çıkan Özellikler
Müşteri Kaydı: Yeni müşteri oluşturma ve mevcut bilgileri yönetme.

Hesap Türleri: Müşterilerin Vadeli veya Vadesiz hesap seçebilmesi.

Faiz Hesaplama: Vadeli hesaplar için sistem tarafından otomatik uygulanan faiz oranları ve getiri hesaplamaları.

Finansal İşlemler: Para çekme, para yatırma ve hesaplar arası transfer işlemleri.

İşlem Geçmişi: Tüm hareketlerin MS SQL veritabanında güvenli bir şekilde saklanması.

# Kullanılan Teknolojiler
Dil: C#
Veritabanı: MS SQL Server
Platform: Windows Forms (WinForms)

# 📂 Veritabanı Kurulumu
Veritabanı yapısını kendi SQL Server'ınızda oluşturmak için database/ klasörü altındaki banka_otomosyonu.sql dosyasını kullanabilirsiniz.
